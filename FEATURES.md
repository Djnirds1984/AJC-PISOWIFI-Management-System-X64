# AJC PISOWIFI Management System - Features

This document outlines the comprehensive features and capabilities of the AJC PISOWIFI Management System.

## 1. Dashboard & Analytics
- **Real-time Monitoring**: View active users, CPU usage, memory usage, and system uptime.
- **Sales Analytics**: Track income from coin slots (Daily, Weekly, Monthly, Total).
- **Session Tracking**: Monitor active sessions, remaining time, and data usage.

## 2. Hotspot Management
- **Captive Portal**: 
  - Customizable landing page.
  - Automatic detection for Apple (CNA) and Android devices.
  - "Insert Coin" workflow integration.
- **Rates Management**: 
  - Configure time-based rates (e.g., 1 Peso = 10 Minutes).
  - Flexible pricing tiers.
- **Bandwidth Management**:
  - Global speed limits.
  - Per-user speed limits (Upload/Download).
- **Session Control**:
  - Pause/Resume time functionality.
  - Auto-disconnect on time expiration.
- **Gaming Priority**: 
  - Dedicated traffic prioritization for low-latency gaming.
- **Device Management**:
  - MAC Address Whitelisting (Bypass portal).
  - MAC Address Blocking (Blacklist).

## 3. PPPoE Server (ISP Management)
- **Built-in PPPoE Server**: 
  - Fully integrated PPPoE server for managing wired clients.
  - Isolated database (`pppoe.sqlite`) for stability.
- **User Management**:
  - Create, Edit, Delete PPPoE users.
  - Set expiration dates for accounts.
- **Expiration Actions**:
  - **Disable**: Automatically disable account upon expiration.
  - **Redirect**: Redirect user to a specific URL (e.g., payment portal) when expired.
- **Billing Profiles**:
  - Create subscription plans with specific speeds and prices.
- **Speed Limiting**:
  - Enforce specific bandwidth limits per PPPoE profile.

## 4. Network Configuration
- **Interface Management**: 
  - Configure LAN and WAN interfaces.
  - Automatic interface detection.
- **Wireless AP**: 
  - Configure built-in WiFi Access Point (SSID, Password).
- **VLAN Support**: 
  - Create and manage VLANs (802.1Q) for network segmentation.
- **Bridge Support**: 
  - Create network bridges to combine interfaces.
- **Multi-WAN**: 
  - Load balancing and failover support (PCC/ECMP modes).

## 5. Hardware Integration
- **Coin Slot Support**: 
  - Direct GPIO integration for coin acceptors.
  - **NodeMCU Integration**: Wireless coin slot extenders via ESP8266/ESP32.
- **NodeMCU Manager**:
  - Manage multiple NodeMCU devices.
  - Over-the-Air (OTA) firmware updates for NodeMCUs.
  - USB Flashing tool built-in.
- **License System**:
  - Hardware-locked licensing system (Trial and Activated modes).

## 6. System Management
- **Backup & Restore**: 
  - Full system configuration backup to `.nxs` file.
  - One-click restore functionality.
- **System Updates**: 
  - Upload-based system updater.
- **Cloud Sync (EdgeSync)**: 
  - Remote status monitoring and synchronization.
- **ZeroTier VPN**: 
  - Built-in ZeroTier integration for secure remote access without port forwarding.
- **Chat System**:
  - Integrated chat for communication between Admin and Users.
- **Logs**:
  - View system logs, access logs, and error logs for troubleshooting.

## 7. Security
- **Admin Authentication**: Secure login for management dashboard.
- **Firewall**: Built-in iptables firewall rules for traffic control.
- **Anti-Hacking**:
